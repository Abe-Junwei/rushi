# 如是我闻 — 简版用户说明

本文件随安装包放在应用 **Resources** 目录（与 `user-guide-zh.pdf` 同目录）。完整开发与架构说明见仓库 `README.md` 与 `docs/`。

## 1. 首次使用

1. 启动桌面应用后，数据目录在系统「应用数据」下（诊断 zip 中的 `build-info.txt` 会写明路径）。
2. **转写不在桌面内跑**：需在本机另开终端启动 ASR 服务（默认 `http://127.0.0.1:8741`）。详见仓库 `services/asr/README.md`。
3. 未安装 FunASR 或未配置模型时，ASR 为 **stub**（占位），识别结果可能为空；要中文识别请按该 README 安装并设置 `RUSHI_FUNASR_MODEL`。

## 2. 项目与转写

1. 创建项目并选择本地音频文件。
2. 确认 ASR 已启动后，在界面中执行「拉取 / 转写」。
3. 可在界面中编辑语段、导出文本或 Word（若版本已提供导出功能）。

## 3. 排障

1. 使用界面中的 **导出诊断 zip**（若已提供）：内含版本信息、可选数据库副本、最近编辑流水与 `logs` 下日志尾部。
2. 检查 ASR 终端日志与 `GET /health` 是否正常。
