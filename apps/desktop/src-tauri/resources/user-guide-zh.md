# 如是我闻 — 简版用户说明

本文件随安装包放在应用 **Resources** 目录。应用内更完整的流程说明见 **设置 → 使用说明**；完整开发与架构说明见仓库 `README.md` 与 `docs/`。

## 1. 首次使用

1. 启动桌面应用后，数据目录在系统「应用数据」下（导出诊断 zip 时 `build-info.txt` 会写明路径）。
2. **本机转写**：安装包会尝试自动拉起 bundled ASR 侧车（默认 `http://127.0.0.1:8741`）。若未就绪，请打开 **设置 → 本机 ASR**，按 **LocalAsrSetupWizard** 完成模型下载与「应用并重启侧车」。
3. **在线转写**（可选）：在 **设置 → 在线 STT** 配置厂商 API Key 并探测通过。
4. 版本号、第三方组件许可见 **设置 → 关于**（v1.1 起提供；此前版本见诊断 zip 内 `build-info.txt`）。

## 2. 项目与转写

1. 在欢迎页 **新建项目** 并选择本地音频文件；可在 **项目信息** 中填写讲述人、时间、地点等场次元信息（可选，导出 Word 时可带入）。
2. 在编辑器工具栏选择 **本机** 或 **在线**，点击 **自动转录**。
3. 编辑语段后 **保存到 SQLite**；可通过工具栏导出 TXT / SRT / DOCX，或使用 **交付导出** 向导导出讲稿、逐字稿等形态。

## 3. 排障

1. 使用 **导出诊断包（zip）**（编辑器菜单或环境页）：内含版本、平台、数据路径、转写时间线摘要与日志尾部。
2. 转写失败时查看编辑器内 **转写诊断** 横幅，或环境页 **本机 ASR** 三盏灯状态。
3. 本机 ASR 问题优先在 **设置 → 本机 ASR** 重试侧车、核对模型是否与侧车一致（「应用并重启侧车」）。

## 4. 许可

第三方组件（含 ffmpeg 等）许可全文见 **设置 → 关于**；软件许可以仓库 `LICENSE` 为准。

## 5. 键盘快捷键

完整键位表见应用内 **设置 → 环境 → 编辑器快捷键**（与右键菜单、底栏提示一致）。常用项：

- **⌘,**（Ctrl+,）：打开设置
- **⌘F**：查找替换
- **⌘J / ⌘K**：与下一条 / 上一条合并
- **⌘D**：在播放头位置拆分语段
- **Space / ⇧⌘Space**：播放或暂停（语段正文内请用 ⇧⌘Space，勿用 ⌘Space）
- **Esc**：关闭浮动面板或确认对话框（转写前 gate 对话框 Esc 留在编辑器）

自定义快捷键绑定与命令面板当前版本不提供。
