## §1 — 改字不失焦 + ⌘Z / Ctrl+Z

**操作**

1. 选中语段 A，记住当前正文（例：`原句一`）。
2. 在 **同一语段输入框内** 改字（例：改为 `原句一改`），**不要** 点击其他语段或列表行（保持聚焦）。
3. 停笔，直到底部状态为 **「已自动保存」**（中间可能短暂显示「即将/正在自动保存…」）。
4. 按 **⌘Z**（Windows：**Ctrl+Z**）。

**期望**

- [x] 语段 A 正文恢复为步骤 1 的改字前内容（自动保存写入前）
- [x] （可选）⇧⌘Z / Ctrl+Y redo 正常

**失败时排查**

- 若撤销无反应：是否仍在 IME 组字中；是否未等到「已自动保存」；是否改字时已点过其他语段（会提前 flush）

## §2 — 工具栏「撤销」与 ⌘Z 一致

**操作**

1. 在语段 B 重复 §1 步骤 1–3（改字、等「已自动保存」）。
2. 点编辑器工具栏 **撤销** 按钮（弯箭头），不要按键盘。

**期望**

- [x] 正文与 §1 使用 ⌘Z 的结果一致
- [x] 工具栏重做与 ⇧⌘Z 行为一致

## §3 — 结构操作撤销无退化

> **右键菜单分工**：**先拖选** 再右键 → **「纳入更正记忆…」**（F6+）。无选区时右键（含 WebKit 段末自动选中一字）→ **语段行菜单**（删/并）。也可点时间戳列。

**操作**（各做一小轮即可，不必全做）

| 操作 | 步骤 | 撤销后期望 |
|------|------|------------|
| **删除语段** | 时间戳列或行边距右键 → 删除 → 工具栏 **撤销** | 语段回到列表原位置，正文不变 |
| **合并** | 行右键 → 与上/下一条合并 → **撤销** | 恢复为合并前两条语段 |
| **拆分** | 波形区指针处右键 → 在指针时间拆分 → **撤销** | 恢复为拆分前一条语段 |

**期望**

- [x] 结构操作撤销成功（删/并/拆至少一种）
- [x] 结构撤销后 §1 改字 + 自动保存 + ⌘Z 仍正常
