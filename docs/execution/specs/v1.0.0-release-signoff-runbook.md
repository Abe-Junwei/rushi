## 1. 发版前闸门（两端共用）

```bash
cd /path/to/Rushi
npm run typecheck && npm run test && node scripts/check-architecture-guard.mjs
cd apps/desktop/src-tauri && cargo test --lib && cargo clippy --lib --all-targets
```

确认版本三处均为 `1.0.0`（或目标版本）。

---

## 2. 发版动作

```bash
git push origin main
git tag -a v1.0.0 -m "v1.0.0"
git push origin v1.0.0
```

CI：`tauri-macos` → `tauri-windows` → `verify-cdn-release` 全绿。

## 5. 已知限制（写入对外说明）

| 项 | 说明 |
|----|------|
| mac Gatekeeper | unsigned DMG · 首次需 Control+打开 |
| Win SmartScreen | unsigned NSIS/portable · 用户需确认运行 |
| Win portable | 无应用内更新；推荐 NSIS 首装 |
| Apple / Authenticode | v1.0.0 未配置；不挡功能签收 |
| ASR sidecar 传递依赖 CVE | **2026-07-18 已刷锁**：CPU `pillow==12.3.0` / `setuptools==83.0.0` / `torch==2.13.0`；CUDA lock 同步 pillow/setuptools（torch 仍 `2.11.0+cu126`）。`pip-audit` CPU lock：**No known vulnerabilities** |
