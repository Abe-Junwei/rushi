## §1 — 纠错记忆管理（热词与记忆页）

1. 打开 **热词与记忆** → **纠错记忆** 区。  
2. 确认列表有 **错词→正词**、**命中次数**、**稳定/采纳** 状态。  
3. 搜索框过滤一条。  
4. 勾选 1～2 条 → **采纳为规则** → 状态变为已采纳。  
5. 勾选另一条 → **删除** → 列表消失。

**期望**

- [ ] 文案无 SQLite/技术库用语  
- [ ] 删除后 F1 预览不再使用该对（若已无其他稳定规则可跳过）

## §2 — 挖掘推荐（LEX-MINE-1）

**准备**：有一条 **命中 ≥3 或已采纳**、且 **术语表尚无该正形** 的记忆（若 MEM-P0 已自动进表，可新建：`错A→对A` 并采纳为规则，或手动删术语表里的 term 再测）。

1. 滚动到 **推荐加入术语表**。  
2. 应看到候选（正形 + 命中次数 + 示例错形）。  
3. **加入术语表** 一条 → 术语列表出现该 term；推荐区消失或刷新后不再出现。  
4. 再准备一条候选 → **忽略** → 刷新后不再出现。

**期望**

- [ ] 批量 **采纳选中** / **忽略选中** 可用  
- [ ] 加入术语表 **不会** 把错形写入别名（热词仅正形）

## §3 — F6 纳入文案（spot-check）

1. 语段选区 → 右键 **纳入更正记忆…**  
2. 读对话框说明（应提到保存计次、满 3 次进术语表）。

**期望**

- [ ] 与 MEM-P0 规则一致
