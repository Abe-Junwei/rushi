## P5′ 全局条与选中行为

| # | 操作 | 预期 |
|---|------|------|
| 1 | 打开编辑器 | 主波形可调高度；全局条 **展开** |
| 2 | 点「折叠」 | 全局条收起；主波形区不变 |
| 3 | 切换项目内另一音频文件 | 全局条 **自动展开** |
| 4 | 准星 **关**，点列表/波形选语段 | px/s **不变**；视口滚到语段 |
| 5 | 准星 **开**，选语段 | 语段 fit 视口 |
| 5b | 准星 **开**，连续键盘/列表切换相近长度语段 | 应 **流畅**（多数仅滚动，无整段 ws.load 卡顿） |
| 5c | **手动 ↔ 跟随语段** 快速切换（有选中语段） | 无长时间卡顿；peaks **不空白**；切手动不强制缩回 100% |
| 5d | 跟随语段已开，再点「跟随语段」 | 仅重定位当前语段（不重复 force 放大） |
| 6 | 点准星开启（有选中） | 立即 fit 当前语段 |
| 7 | 全局条：拖视口框 / 点空白 / 语段 chip | 与 P4 minimap 一致 |
| 8 | 主波形横向滚动后 | peaks 与 overlay 对齐，不空白 |

## P5′ 缩放与导航模式（能力—UI 对齐）

| # | 操作 | 预期 |
|---|------|------|
| 1 | 分段控件选「跟随语段」 | footer「波形：跟随语段」；换语段视口跟随（能放下只滚） |
| 2 | 分段控件选「手动」 | footer「波形：手动缩放」；滑块可拖；不自动 fit 语段 |
| 3 | 跟随 + 长语段 | px/s 可低于 16；滑块 `belowManualSliderRange` 表现正确 |
| 4 | +/- 达 min/max | 对应按钮 disabled |
| 5 | 全文件一屏 | **全局波形条**拖视口 / 点空白 seek，非主工具栏按钮 |
| 6 | 跟随 ON，peaks 就绪后选语段 | 主波形 Canvas 有峰（非空白） |
| 7 | 点列表/波形选语段 | 播放头在 **语段起点**，非中点 |
| 8 | 播放区输入 `12:30` Enter | 跳转到对应时间；非法格式有 hint |
| 9 | 折叠全局条并播放 | **主波形** playhead 仍移动；展开时 minimap 竖线同步 |
| 10 | Tab 下一段（默认） | 用 **全局变速** 播放；语段 **循环** 开启 |
| 11 | 跟随模式连续换相近长度语段 | 多数仅滚动；px/s 按 **8px/s 分档** 少 reload |
