# 调研：R3h-3.5 — Sherpa-ONNX Paraformer Spike（替代 SenseVoice 评估轨）

> **状态**：已采纳（2026-06-10）  
> **关联路线图**：[`rushi-execution-roadmap.md`](../plans/rushi-execution-roadmap.md) §4.1.1 **⑧½**  
> **关联 ADR**：[ADR-0003](../../adr/0003-asr-engine-funasr-first-sherpa-spike-gate.md)（附录 A 已按本文修订）  
> **关联验收**：[`r3h-3.5-sherpa-spike-acceptance.md`](./r3h-3.5-sherpa-spike-acceptance.md)

## 1. 问题陈述

| 项 | 内容 |
|----|------|
| **用户场景** | 个人 v1 需「零终端」本机转写；当前 PyInstaller FunASR 侧车 ~2.5GB，弱网下载与冷启动成本高 |
| **本仓现状** | **唯一 catalog 主推 SKU** = Paraformer 长音频（`localAsrModelCatalog.ts`）；`SenseVoiceSmall` **已弃用**（`DEPRECATED_LOCAL_ASR_HUB_MODEL_IDS`）；Qwen3-ASR **No-go**（Q-ASR-1）；LRC manifest/installer **引擎无关** |
| **成功标准** | Sherpa-ONNX **Paraformer** 路径在 CER、13min 语段、时间戳、hotwords 上与 FunASR 侧车对比后产出 **Go / Partial Go / No-go**；不删除 FunASR 主路径 |

**范围校正（2026-06-10）**：旧 ADR/remediation 以 **SenseVoice INT8** 为 Spike 主模型；与 **Q-ASR-1 / catalog 真源** 冲突。**本 Spike 以 Paraformer 对齐 v1 默认 SKU**，SenseVoice 仅作历史基线对照（可选），不作为 Go 门控对象。

## 2. 业内成熟路线（≥2）

| # | 路线 | 代表 | 核心机制 | 链接 |
|---|------|------|----------|------|
| A | **保持 FunASR + PyInstaller 侧车** | Rushi 当前 | HTTP 8741、`model_prepare`、VAD+punc+`sentence_timestamp` | `services/asr/rushi_asr/` |
| B | **Sherpa-ONNX + Rust crate** | k2-fsa/sherpa-onnx | 静态链接 ONNX Runtime；离线/流式 Paraformer、Zipformer；官方 Rust 示例 | [sherpa-onnx](https://github.com/k2-fsa/sherpa-onnx)、[Paraformer 模型表](https://k2-fsa.github.io/sherpa/onnx/pretrained_models/offline-paraformer/paraformer-models.html) |
| C | **faster-whisper / whisper.cpp** | whisper.cpp、CTranslate2 | 英文强、中文 CER 劣化；无 Paraformer 长音频生态 | remediation **不采纳** |

## 3. 可复用评估

| 路线 | 复用度 | 可直接用 | 与 Rushi 约束冲突 | 进度 / 体积 |
|------|--------|----------|-------------------|-------------|
| A FunASR 侧车 | **高** | 全链路、hotwords、R3t 分段、eval | ~2.5GB 体积 | 已签收 |
| B Sherpa Paraformer | **中** | Rust 内嵌、~100–300MB 级运行时+权重 | FunASR SKU 为 **vad-punc 一体**；Sherpa 多为 **离线 Paraformer**，VAD/punc/时间戳需 **拼装或换官方 VAD+ASR 包**；**hotwords** 待验证 | Spike 目标 |
| C whisper 系 | **低** | — | 中文精度、SenseVoice/Paraformer 能力不匹配 | — |

**本仓已有可复用模块（须扩展而非 fork）**：

- 分段真源：`services/asr/rushi_asr/segmentation.py`（R3t-A）
- 评估样本：`docs/execution/fixtures/asr-hotword-eval/`、R3t 13min 手测素材路径
- LRC：`apps/desktop/src-tauri/src/local_runtime/`（引擎无关）
- 能力对齐：[`desktop-capability-ui-state-alignment.md`](../../architecture/desktop-capability-ui-state-alignment.md)

## 4. 决策摘要

| 问题 | 结论 |
|------|------|
| **Spike 主模型** | Sherpa **`paraformer-zh-2023-09-14`**（支持 timestamps）+ **`paraformer-zh-int8-2025-10-07`**（体积/RTF）；基线 = FunASR **`speech_paraformer-large-vad-punc_...`** |
| **不测 / 不作 Go 依据** | SenseVoice（弃用）、Qwen3-ASR（No-go）、faster-whisper、纯 whisper.cpp |
| **不做什么** | Spike 周内删除 Python 侧车；产品级双引擎切换 UI；LRC manifest 大改 |
| **与 ADR** | 维持方案 A；结论写入 `docs/adr/00xx-sherpa-onnx-evaluation.md` 或 ADR-0003 附录 B |
| **风险** | Sherpa Paraformer ≠ FunASR paraformer-large-**vad-punc** 同构；语段/标点/hotwords 差距可能导致 **Partial Go** 或 **No-go** |

### Sherpa 候选模型（Spike 真源）

| 优先级 | 模型 ID | 用途 |
|--------|---------|------|
| P0 | `sherpa-onnx-paraformer-zh-2023-09-14` | 主对比；官方支持 **timestamps** |
| P1 | `sherpa-onnx-paraformer-zh-int8-2025-10-07` | 轻量 / RTF |
| P2 | Sherpa **VAD + Paraformer-large** 预训练包 | 逼近 FunASR vad-punc 管线 |
| 雷达 | `zipformer-ctc-zh-int8-2025-07-03` | 极小体积；多半无标点/弱时间戳 |
