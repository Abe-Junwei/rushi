## 环境

- [ ] `npm run desktop:dev`（或已安装构建）
- [ ] 打开 **≥50 语段** 且含 **纠错记忆/规则** 的文件（错词镜像会启用）
- [ ] 备选：D3 对照项目 ~185 段 + 术语/记忆若干条
- [ ] 录制可选：Chrome Performance 5s 连续输入（对比 P0 前后）

---

## H1 — 连续输入跟手（有错词规则）

| 步 | 操作 | 期望 |
|----|------|------|
| 1 | 选中中段一条较长语段，点正文 | 进入 textarea，光标正常 |
| 2 | 连续快速输入 20–30 字（拼音或直接字） | **无** 整行闪白/透明字消失；光标处 **始终看见黑色/实字** |
| 3 | 暂停 0.5s | 错词虚线 **出现/更新**（可略滞后于最后一键） |
| 4 | 继续输入 | 仍跟手，无「打一字卡一下」 |

- [x] H1 通过

**回归**：无纠错规则的项目输入同样顺滑（镜像关闭）。

## H2 — 查找替换高亮仍跟手

## H4 — 页脚字数

| 步 | 操作 | 期望 |
|----|------|------|
| 1 | 记当前底栏 **字数** | — |
| 2 | 在当前段追加 10 字，**不停顿** 看底栏 | 数字可在输入中 **略滞后** |
| 3 | 停 0.3s | 字数 **与稿内总字一致**（±0） |

## H5 — 持久化与语义不变

| 步 | 操作 | 期望 |
|----|------|------|
| 1 | 编辑后 **blur**（点他处） | 语段文本更新；未保存点行为不变 |
| 2 | 等自动保存「已自动保存」 | SQLite 落库 |
| 3 | 关闭文件再打开 | 文本一致 |
| 4 | ⌘Z 撤销一步 | 与 P0 前一致 |
| 5 | Tab 跳下一段 | 仍可编辑（P0 **不要求** 改滚动策略） |

- [x] H5 通过
