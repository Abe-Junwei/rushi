# Stitch 上传包（集中目录）

本目录用于集中存放后续需要上传给 Stitch 的资料。

## 使用方式

在仓库根目录执行：

```bash
bash scripts/prepare-stitch-upload.sh
```

执行后会将最新资料同步到本目录。
